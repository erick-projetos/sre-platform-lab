# Microservices

Esta pasta contém os microserviços utilizados no laboratório SRE.

Os serviços simulam uma arquitetura simples de aplicações distribuídas.

## Serviços

auth-service  
Responsável por autenticação de usuários.

analytics-service  
Responsável pelo processamento de eventos e geração de métricas.

## Objetivo

Esses serviços expõem métricas que são coletadas pelo Prometheus para simular um ambiente real de observabilidade de microserviços.
