# Docker 

Esta pasta contém a configuração da infraestrutura do laboratório utilizando Docker Compose.

## docker-compose

O arquivo docker-compose.yml define todos os serviços executados no ambiente.

## Serviços do ambiente

- Microservices
- Prometheus
- Grafana
- Alertmanager
- Node Exporter
- cAdvisor
- Loki
- Promtail

## Objetivo

Permitir a execução de todo o laboratório localmente utilizando containers Docker.
