# Monitoring Stack

Esta pasta contém as configurações da stack de monitoramento utilizada no laboratório SRE.

## Ferramentas utilizadas

Prometheus  
Responsável por coletar métricas dos serviços e da infraestrutura.

Grafana  
Responsável pela visualização das métricas através de dashboards.

Alertmanager  
Responsável por gerenciar alertas gerados pelo Prometheus.

## Objetivo

Simular um ambiente de observabilidade semelhante ao utilizado em sistemas distribuídos modernos.
